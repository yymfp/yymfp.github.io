  • cookie

    дё»иҰҒз”ЁжқҘи®°еҪ•з”ЁжҲ·иә«д»ҪдҝЎжҒҜпјҢеӣ дёәHTTPжҳҜж— зҠ¶жҖҒзҡ„еә”з”ЁеұӮеҚҸи®®пјҢжңҚеҠЎеҷЁдёҚдјҡи®°дҪҸе®ўжҲ·з«Ҝжң¬ж¬Ўзҡ„иҜ·жұӮж“ҚдҪңгҖӮ

    еӨ§е°Ҹпјҡ4kb

    зү№зӮ№пјҡж— и®әиҜ·жұӮжҳҜеҗҰз”Ёеҫ—еҲ°пјҢйғҪдјҡиў«жқҘеӣһдј йҖ’гҖӮеҸҜд»Ҙи®ҫзҪ®иҝҮжңҹж—¶й—ҙ

    е®үе…ЁжҖ§пјҡjsи„ҡжң¬еҸҜд»ҘйҖҡиҝҮdocument.cookieиҺ·еҸ–жҲ–иҖ…и®ҫзҪ®cookieгҖӮйҖҡиҝҮи®ҫзҪ®cookieж—¶пјҢж·»еҠ httpOnlyзҰҒжӯўjsиҜ»еҸ–пјҢд»ҺиҖҢеҸҜд»Ҙйў„йҳІxssж”»еҮ»гҖӮпјҲдёҚиҰҒз”Ёcookieдј йҖ’ж•Ҹж„ҹдҝЎжҒҜпјү

    е…ұеҗҢзӮ№пјҡз”ЁдәҺжөҸи§ҲеҷЁз«ҜеӯҳеӮЁзҡ„зј“еӯҳж•°жҚ®,йғҪйҒөеҫӘеҗҢжәҗзӯ–з•ҘгҖӮ

    дёҚеҗҢзӮ№пјҡ

    1. еӯҳеӮЁеҶ…е®№жҳҜеҗҰеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁз«ҜпјҡеҪ“и®ҫзҪ®дәҶCookieеҗҺпјҢж•°жҚ®дјҡеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁз«ҜпјҢйҖ жҲҗдёҖе®ҡзҡ„е®ҪеёҰжөӘиҙ№пјӣweb storage,дјҡе°Ҷж•°жҚ®дҝқеӯҳеҲ°жң¬ең°пјҢдёҚдјҡйҖ жҲҗе®ҪеёҰжөӘиҙ№пјӣ
    2. ж•°жҚ®еӯҳеӮЁеӨ§е°ҸдёҚеҗҢпјҡCookieж•°жҚ®дёҚиғҪи¶…иҝҮ4K,йҖӮз”ЁдәҺдјҡиҜқж ҮиҜҶпјӣweb storageж•°жҚ®еӯҳеӮЁеҸҜд»ҘиҫҫеҲ°5M;
    3. ж•°жҚ®еӯҳеӮЁзҡ„жңүж•ҲжңҹйҷҗдёҚеҗҢпјҡcookieеҸӘеңЁи®ҫзҪ®дәҶCookidиҝҮжңҹж—¶й—ҙд№ӢеүҚдёҖзӣҙжңүж•ҲпјҢеҚіе…ій—ӯзӘ—еҸЈжҲ–иҖ…жөҸи§ҲеҷЁпјӣsessionStorage,д»…еңЁе…ій—ӯжөҸи§ҲеҷЁд№ӢеүҚжңүж•ҲпјӣlocalStorage,йҷӨйқһжүӢеҠЁеҲ йҷӨпјҢеҗҰеҲҷж•°жҚ®еӯҳеӮЁж°ёд№…жңүж•Ҳпјӣ
    4. дҪңз”ЁеҹҹдёҚеҗҢпјҡcookieе’ҢlocalStorageжҳҜеңЁеҗҢжәҗдёӯе…ұдә«зҡ„пјҢеҚіеҸӘиҰҒеңЁзӣёеҗҢеҚҸи®®гҖҒзӣёеҗҢдё»жңәеҗҚгҖҒзӣёеҗҢз«ҜеҸЈдёӢпјӣsessionStorageйҷӨдәҶеҗҢжәҗеӨ–пјҢиҝҳеҝ…йЎ»жҳҜеҗҢдёҖдёӘзӘ—еҸЈпјҲд№ҹе°ұжҳҜжөҸи§ҲеҷЁзҡ„ж ҮзӯҫйЎөпјүпјӣ

  • Web StorageдёҺCookieзӣёжҜ”еӯҳеңЁзҡ„дјҳеҠҝпјҡ

    1. еӯҳеӮЁз©әй—ҙжӣҙеӨ§пјҡIE8дёӢжҜҸдёӘзӢ¬з«Ӣзҡ„еӯҳеӮЁз©әй—ҙдёә10MпјҢе…¶д»–жөҸи§ҲеҷЁе®һзҺ°з•ҘжңүдёҚеҗҢпјҢдҪҶйғҪжҜ”CookieиҰҒеӨ§еҫҲеӨҡгҖӮ

    2. еӯҳеӮЁеҶ…е®№дёҚдјҡеҸ‘йҖҒеҲ°жңҚеҠЎеҷЁпјҡеҪ“и®ҫзҪ®дәҶCookieеҗҺпјҢCookieзҡ„еҶ…е®№дјҡйҡҸзқҖиҜ·жұӮдёҖе№¶еҸ‘йҖҒзҡ„жңҚеҠЎеҷЁпјҢиҝҷеҜ№дәҺжң¬ең°еӯҳеӮЁзҡ„ж•°жҚ®жҳҜдёҖз§ҚеёҰе®ҪжөӘиҙ№гҖӮиҖҢWeb Storageдёӯзҡ„ж•°жҚ®еҲҷд»…д»…жҳҜеӯҳеңЁжң¬

      ең°пјҢдёҚдјҡдёҺжңҚеҠЎеҷЁеҸ‘з”ҹд»»дҪ•дәӨдә’гҖӮ

    3. жӣҙеӨҡдё°еҜҢжҳ“з”Ёзҡ„жҺҘеҸЈпјҡWeb StorageжҸҗдҫӣдәҶдёҖеҘ—жӣҙдёәдё°еҜҢзҡ„жҺҘеҸЈпјҢеҰӮsetItem,getItem,removeItem,clearзӯү,дҪҝеҫ—ж•°жҚ®ж“ҚдҪңжӣҙдёәз®ҖдҫҝгҖӮcookieйңҖиҰҒиҮӘе·ұе°ҒиЈ…гҖӮ

    4. зӢ¬з«Ӣзҡ„еӯҳеӮЁз©әй—ҙпјҡжҜҸдёӘеҹҹпјҲеҢ…жӢ¬еӯҗеҹҹпјүжңүзӢ¬з«Ӣзҡ„еӯҳеӮЁз©әй—ҙпјҢеҗ„дёӘеӯҳеӮЁз©әй—ҙжҳҜе®Ңе…ЁзӢ¬з«Ӣзҡ„пјҢеӣ жӯӨдёҚдјҡйҖ жҲҗж•°жҚ®ж··д№ұгҖӮ

  • дҪҝз”ЁsessionStorageйҒҮеҲ°зҡ„еқ‘

    1. sessionStorageжңүиҰҒжұӮеҝ…йЎ»жҳҜеҗҢдёҖдёӘжөҸи§ҲеҷЁж ҮзӯҫtabдёӢзҡ„жүҚеҸҜд»Ҙи®ҝй—®гҖӮ

      дҪҶе®һйҷ…еңЁиҝҗз”ЁдёӯеҸ‘зҺ°пјҢйҖҡиҝҮйЎөйқўеҶ…зҡ„й“ҫжҺҘпјҲжҲ–иҖ…з”Ёwindow.openпјүжү“ејҖж–°зҡ„ж ҮзӯҫйЎөпјҢе®ғд»¬д№Ӣй—ҙд№ҹжҳҜеҸҜд»ҘиҜ»еҸ–еҲ°sessionзҡ„пјҲдҪҶж„ҹи§үдёҚжҳҜе…ұдә«зҡ„пјҢдҝ®ж”№дәҶж–°ејҖйЎөзҡ„sessionеҖјпјҢеҺҹжқҘзҡ„е№¶дёҚеҸ—еҪұе“ҚпјүгҖӮиҖҢйҖҡиҝҮжөҸи§ҲеҷЁзӣҙжҺҘж–°ејҖдёҖдёӘtabжҳҜпјҢеҚідҪҝзҪ‘з«ҷдёҖж ·пјҢд№ҹжҳҜиҜ»еҸ–дёҚеҲ°зҡ„гҖӮ
